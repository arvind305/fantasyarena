# UNIVERSAL FRIENDS PREDICTION GAME — CONSTITUTION

## VERSION
2.0 — Sport-Agnostic Generalization

## SECTION 1 — OBJECTIVE & DESIGN PHILOSOPHY
The game is a private, non-monetary, skill-plus-luck based prediction game for any competitive event. It maximizes engagement, unpredictability, and fairness.

## SECTION 2 — EVENTS
An Event represents a real-world contest. Events are centrally created by Admin based on official schedules.

## SECTION 3 — BETTING COMPONENTS
Admins define event-specific betting questions. Users only respond to predefined bets.

## SECTION 4 — SCORING
Each Ruleset defines scoring logic. Base scoring rules cannot be altered mid-event.

## SECTION 5 — MULTIPLIERS
Optional volatility mechanics applied only to entity-based predictions.

## SECTION 6 — RUNNERS / PROXIES
Optional mechanism where another user’s score partially contributes.

## SECTION 7 — SIDE BETS
Optional, unique, clearly defined bets per event.

## SECTION 8 — NEGATIVE SCORING
May apply to non-entity bets only.

## SECTION 9 — LOCKING
All predictions lock at event start.

## SECTION 10 — GROUPS
Groups are leaderboard overlays only. Global group includes all users.

## SECTION 11 — ADMIN POWERS
Admins may configure events and bets but may not alter locked user predictions.

## SECTION 12 — DATA SOURCES
Read-only official data sources.

## SECTION 13 — ANALYTICS
Transparent, read-only analytics at user, group, and event levels.

## SECTION 14 — ARCHITECTURE
Frontend, Backend API, Rules Engine, Scoring Engine, Data Ingestion.

## SECTION 15 — FINALITY
This constitution is authoritative unless versioned.

END OF CONSTITUTION
