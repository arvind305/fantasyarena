## INTENT SUMMARY

This batch covers tournament-level administration. The Admin Panel follows strict design principles:

- **Muted palette** (darker base, blue accent instead of cyan)
- **No animations or glows** (serious operational context)
- **Tiered action buttons** by severity (green/yellow/red)
- **Audit trail with timestamps** for every change
- **Type-to-confirm** for destructive actions
- **Read-before-write pattern** (review state before changing)

**Key Screens:**
- Admin Dashboard
- Match Configuration
- Side Bet Management
- Side Bet Reveal (Live)
- Score Entry & Finalization
- Correction Flow
- Audit Log
- Match Abandonment
